FAQ¶

IRIS BigData solution을 활용하며 발생하는 다양한 종류의 Question과 그에 대한 답변 / 사용상의 팁을 모아놓은 저장소입니다.

Q. 내가 소유한 데이터를 서비스 무료 체험에서 사용할 수 있나요?¶

예, 가능합니다.
그러나, 무료 체험을 위해서는 제한된 크기의 자원이 할당됩니다. 무료 체험 사용자에게 제공되는 자원의 사양은 다음과 같습니다.

  • 최대 100GB의 데이터 저장공간 제공

무료 체험 사용자가 소유한 데이터를 분석하기 위해서는 기본적으로 데이터 연결 및 데이터 모델을 생성해야 합니다. 데이터 연결 및 데이터 모델의 생성 방법은 아래 메뉴얼을 참조하시기 바랍니다.

  • 데이터 업로드 및 데이터 모델 생성하기: <http://docs.iris.tools/manual/IRIS-Tutorial/Getting-Started/01-intro/index.html>

  • 데이터 연결하기: <http://docs.iris.tools/manual/IRIS-Manual/IRIS-Common/inquiry_management/connect_info/index.html>


Q. 우리 회사에서 여러 명이 무료 체험 서비스를 이용하려고 합니다. 각자 별도의 도메인을 입력해야 하나요?¶

그렇지 않습니다.
일반적으로, 영문 회사명의 도메인으로 무료체험을 신청하면, 신청한 사용자가 관리자(root) 권한을 갖게 됩니다. 일단 해당 도메인의 무료 체험 계정이 만들어지면, 회사 내의 다른 사용자는 해당 도메인에서 가입을 통해 일반 사용자로 무료 체험을 사용할 수 있습니다.
도메인 생성 후, 일반 사용자 계정 생성 방법은 아래 메뉴얼을 참조하시기 바랍니다.

  • 일반 사용자 계정 생성 방법: <http://docs.iris.tools/manual/IRIS-Manual/IRIS-Common/inquiry_management/user_management/index.html>


Q. 서비스 무료 체험 시한이 곧 만료됩니다. 사내에서 유료 전환을 위한 결제절차가 지연되고 있습니다. 무료 체험 시한이 만료된 후, 유료 전환하지 않으면 무료 체험기간동안 작성한 보고서는 어떻게 되나요?¶

무료 체험 시한이 만료되고 유료 서비스로 전환하지 않은 경우, 무료 체험 기간중에 사용자가 작성한 모든 데이터는 서비스 이용 약관에 따라 1개월동안 보관 후에 소멸됩니다. 따라서, 무료 체험 시한이 만료되더라도 1개월 내에 유료 서비스로 전환하시면 기존 데이터 및 보고서를 계속 사용할 수 있습니다.
무료 체험 시한과 관련한 기타 문의사항은 아래에서 문의하시기 바랍니다.

  • 문의하기: <http://www.iris.tools/views/product/product-inquiry>


Q. 서비스 처음 사용자입니다. 서비스를 이용하려면 어떻게 해야 하나요?¶

서비스를 처음 사용하려면 IRIS 따라하기 <http://docs.iris.tools/manual/IRIS-Tutorial/Getting-Started/index.html>의 안내를 참조하시기 바랍니다.

데이터 연결 방법은 다음 메뉴얼을 참조하시기 바랍니다.

  • 데이터 연결하기: <http://docs.iris.tools/manual/IRIS-Manual/IRIS-Common/inquiry_management/connect_info/index.html>

  • 데이터 모델 생성하기: <http://docs.iris.tools/manual/IRIS-Tutorial/Getting-Started/01-intro/index.html#id3>


Q. 내가 IRIS SaaS에서 사용하는 데이터를 다른 사용자가 접근할 수 있도록 할 수 있나요?¶

기본적으로 개별 사용자가 업로드하거나 연결한 데이터는 해당 사용자외에는 사용할 수 없습니다.
그러나, 동일한 도메인의 사용자간에는 타 사용자가 내 데이터에 접근할 수 있도록 접근권한을 설정할 수 있습니다.
데이터 접근 권한을 설정하는 방법은 다음 메뉴얼을 참조하시기 바랍니다.

Q. 내가 IRIS SaaS에서 작성한 보고서를 다른 사용자와 공유하려면 어떻게 해야 하나요?¶

내가 작성한 보고서는 다음의 두 가지 방법으로 타 사용자와 공유할 수 있습니다.
1. 보고서 접근권한 설정
동일한 도메인의 타 사용자 혹은 사용자 그룹에게 내가 작성한 보고서의 접근 권한을 허용할 수 있습니다.
접근이 허용된 사용자 혹은 사용자 그룹의 사용자들은 해당 보고서를 복사하여 수정할 수 있습니다. 사용자간 보고서 접근 권한 설정 방법은 다음 메뉴얼을 참조하시기 바랍니다.

  • 보고서 접근 권한 설정 방법: <http://docs.iris.tools/manual/IRIS-Manual/IRIS-Studio/start.html#id2>

2. 보고서 공유
동일한 도몌인 혹은 타 도메인(외부 사용자)에게 내가 작성한 보고서의 URL을 이용하여 공유할 수 있습니다. 이 경우, 공유 받은 보고서는 편집이 불가능하며, 실행만 가능합니다.
보고서의 공유 방법은 다음 메뉴얼을 참조하시기 바랍니다.

  • 타 사용자 혹은 외부 사용자와 보고서 공유 방법: <http://docs.iris.tools/manual/IRIS-Manual/IRIS-Studio/start.html#id2>

Q. 동일한 사용자 ID로 여러 사용자가 동시에 서비스를 이용할 수 있나요?¶

동일한 사용자 ID로 여러 사용자가 동시에 서비스를 이용할 수 있습니다.

Q. 기존 관리자가 변경되어 관리자 정보를 변경해야 합니다. 어떻게 해야 하나요?¶

관리자 계정(super user)의 ID는 root로 고정되어 있으며 변경이 불가능합니다. 단, 비밀번호를 변경하고자 하는 경우, 관리자 계정으로 로그인 후에 비밀번호 변경이 가능합니다.
비밀 번호의 변경 방법은 다음을 참조하시기 바랍니다.

  • 관리자 계정의 비밀번호 변경 방법: <http://docs.iris.tools/manual/IRIS-Manual/IRIS-Common/inquiry_management/user_management/common.html#id1>


Q. 관리자 계정의 비밀번호가 기억나지 않습니다. 어떻게 해야 하나요?¶

관리자가 도메인을 생성하면서 설정한 관리자 계정의 비밀번호를 제 3자가 확인할 수 있는 기능은 제공하지 않습니다. 따라서, 기존 비밀번호를 초기화한 후에 재설정하여야 합니다.
관리자 계정의 비밀번호를 재설정하려면 아래에서 문의하시기 바랍니다.

  • 문의하기: <http://www.iris.tools/views/product/product-inquiry>


Q. IRIS SaaS에서 사용중인 나의 데이터가 타인에 의해 외부로 유출되지는 않을까요?¶

IRIS SaaS는 각 도메인별로 전용의 가상 네트워크인 VPC(Virtual Private Cloud) 환경을 제공합니다. 따라서, 타인에 의한 데이터의 외부 유출로부터 데이터 보안을 강화하였습니다.

Q. 내 소유의 데이터와 보고서는 어디에 얼마동안 저장되나요?¶

사용자가 작성한 모든 데이터는 서비스 이용 약관에 따라 무료 체험기간 만료 후, 또는 유료 서비스 해지 후 1개월동안 보관 후에 소멸됩니다.

Q. 서비스 이용중, 실수로 데이터/보고서를 삭제했습니다. 삭제한 데이터/보고서를 복원할 수 있나요?¶

삭제된 데이터에 대한 복원 기능은 제공하지 않습니다. 따라서, 데이터 혹은 보고서를 삭제할 경우, 재확인 후에 삭제하시기 바랍니다.
