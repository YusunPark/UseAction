IRIS Studio¶

IRIS Studio는 다양한 데이터 시각화 결과를 웹 문서로 작성할 수 있는 도구를 제공합니다. 분석 결과를 보고서로 작성할 수 있으며 해당 분석기능을 활용하여 데이터소스 및 데이터 탐색 범위, 보고서 갱신 주기 등을 동적으로 변경할 수 있는 대시보드 화면 저작 등에 활용할 수 있습니다.

데이터 설정이 필요한 Studio 객체(챠트, 콤보박스, 체크박스 등) 는 데이터객체 와의 연결로 데이터를 가져오도록 업데이트(2020.07월)되었습니다.
tutorial 내용은 수정중에 있습니다.

데이터 객체에 대한 설명 과 데이터객체를 이용한 챠트 설명 을 참고하시기 바랍니다.
